Difference between map,forEach,filter and find

Map,forEach,filter and find are array method. Mainly map() is used to execute the code then it is given a new array for execution result. On the other hand, forEach() is used to execute same code every element bt it is no returning value. it returns undefined.
Again, filter() is used to checks every element in an array if it fulfill the condition and then it returns a new array for matching element. On the other hand, find() As like as similar of filter but find is used to check condition but it gives only one match element in an array.
